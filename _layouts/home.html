Welcome

As a new employee we want you to be successful in your work. We have developed several training tutorials and instructions to get you started.

Our training program is continualy evolving as is the technology we work with so as new content is available we will publish to this repository and site for your continued education.

Module One

Module 1 Objectives

Upon completion of this module, you will be able to:

  • Define what is gitflow
  • Understand when gitflow is utilized for software development
  • Describe why gitflow is used at CTS-IT

Module Description

This module introduces the concept of Git workflows with specific emphasis on Gitflow. The module will answer why gitflow is used at CTS-IT and it will answer when it is used in our software development lifecycle.
